# 📝 CLI To-Do List Application

A simple command-line To-Do List application built using Python. This project allows users to manage daily tasks efficiently through a menu-driven interface.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed
* ⚠️ Input validation to prevent crashes

---

## 🧠 Concepts Used

* Python functions
* Lists and dictionaries
* Loops (`while`, `for`)
* Exception handling (`try-except`)
* User input handling

---

## 🖥️ How It Works

The program runs in a loop and displays a menu:

1. Add task
2. View task
3. Delete task
4. Mark as done
5. Exit

Users can select an option by entering the corresponding number.

---

## ▶️ How to Run

1. Make sure Python is installed
2. Save the file as `todo.py`
3. Run the program:

```bash
python todo.py
```
